# MNvchad

## clean_nvim.sh

⚠️ Please make sure you fully understand the function of these commands!

```bash
rm -rf ~/.config/nvim \
rm -rf ~/.local/share/nvim \
rm -rf ~/.local/state/nvim \
rm -rf ~/.cache/nvim
```

## Install

```bash
git clone https://github.com/FBamx/nvchad.git -b master ~/.config/nvim
```
